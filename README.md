# Bases

## Dependencias

* Angular 17

* Angular Material
```
npm run ng -- add @angular/material
```

* ngBootstrap
```
npm run ng -- add @ng-bootstrap/ng-bootstrap
```

* uuid
```
npm i uuid
```

```
npm i --save-dev @types/uuid
```

## Generar build de producción
```
npm run build
```

## HTTP Server
```
npm i -D http-server
```
